# Интеграция чата

> [!tip] Доступно на серверах
> 
> **Overnight** · **Sunrise** · **Twilight**

**Bridges** — собственный плагин команды Wilderion, который создаёт «мосты» между чатами разных платформ: Minecraft, Discord и Telegram.

---

## 🌉 Что это?

Bridges объединяет сообщества на разных платформах в единое пространство для общения.

**Что умеет:**
- 💬 **Синхронизация чата** — сообщения из Minecraft отображаются в Discord и Telegram
- 📱 **Обратная связь** — сообщения из Discord и Telegram видны в игре
- 📊 **Статус онлайн** — автоматическое обновление количества игроков онлайн
- 🔔 **Уведомления** — оповещения о запуске/остановке сервера

---

## 💬 Как это работает

### Из Minecraft в Discord/Telegram

Когда вы пишете в чат Minecraft:
1. Сообщение отправляется в игровой чат
2. Автоматически дублируется в Discord канал
3. Автоматически дублируется в Telegram чат

**Формат сообщения:**
```
[Игрок] Привет всем!
```

### Из Discord/Telegram в Minecraft

Когда кто-то пишет в Discord или Telegram:
1. Сообщение отображается в Minecraft чате
2. Игроки видят, откуда пришло сообщение

**Формат сообщения в игре:**
```
[Discord] Пользователь: Привет!
[Telegram] Пользователь: Как дела?
```

---

## 📊 Статус онлайн

Bridges автоматически обновляет информацию об онлайн-игроках:

### В Discord
- Обновление топика канала с количеством онлайн игроков
- Статус бота показывает активность

### В Telegram
- Обновление закреплённого сообщения
- Отображение списка игроков онлайн

---

## 🔔 Системные уведомления

Плагин автоматически отправляет уведомления:

| Событие | Сообщение |
|---------|-----------|
| 🟢 Запуск сервера | «Сервер запущен» |
| 🔴 Остановка сервера | «Сервер остановлен» |
| 👋 Игрок зашёл | «Игрок присоединился к игре» |
| 👋 Игрок вышел | «Игрок покинул игру» |

- 📱 Общайтесь с друзьями, даже не заходя в игру
- 🔔 Получайте уведомления о событиях на сервере

### Для сообщества
- 🌐 Единое пространство для общения
- 📈 Больше вовлечённости игроков

## ❓ Частые вопросы

<details>
<summary>**Видят ли игроки сообщения из Discord/Telegram?**</summary>

Да! Все сообщения из Discord и Telegram отображаются в игровом чате с указанием источника.

</details>

<details>
<summary>**Можно ли отключить синхронизацию?**</summary>

Синхронизация настроена на уровне сервера. Игроки не могут отключить её для себя.

</details>

<details>
<summary>**Есть ли задержка синхронизации?**</summary>

Синхронизация происходит практически мгновенно (обычно менее 1 секунды).

</details>

<details>
<summary>**Работает ли это с приватными сообщениями?**</summary>

Нет, Bridges синхронизирует только публичные каналы чата.

</details>
