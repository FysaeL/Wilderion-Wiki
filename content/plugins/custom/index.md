# Собственные плагины

Плагины, разработанные командой Wilderion для серверов сети.

- [Trust](trust/trust) — защита от гриферства новичков
- [Qolity](qolity/anvil) — улучшения качества жизни (наковальня, элитры, рамки, блоки света, мобы)
- [Art](art/portraits) — портреты на черепах
- [Nexus](nexus/afk) — AFK, ссылки в меню
- [Bridges](bridges/chat-integration) — интеграция чата с Discord и Telegram
- [Hammer](hammer/hammer) — ограничения для игроков
- [Babble](babble/babble) — чат, каналы, форматирование
